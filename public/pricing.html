FAQ

Common questions.

What does self-hosted mean?

You run the Node.js app and SQLite database on hardware you control. Preference text and credentials stay off third-party SaaS unless you choose to use managed hosting elsewhere.

What is MCP?

The Model Context Protocol connects assistants to tools and data. Instill AI exposes your rules through MCP so Claude, Cursor, and compatible clients can pull preferences with one authenticated endpoint.

Do I restart tools after editing preferences?

No. The next time the model calls get_preferences, it receives your latest dashboard state — no redeploy or restart required.

How does ChatGPT fit in?

ChatGPT Custom GPTs use OpenAPI Actions instead of MCP. This server exposes /openapi.json; you import it and pass the same Bearer API key.

Which editors are supported?

See the Integrations page for Cursor, VS Code, Windsurf, Zed, Continue, Claude Code/Desktop, and generic MCP.

Can we change plans later?

If you buy from a vendor who packages Instill AI, they set upgrade paths. If you self-host, you own the config — adjust limits and features in code or environment.

How is this different from “MCP hosting” products?

Hosted gateways often charge for RPC volume, namespaces, or uptime. Instill AI focuses on your preference store and tools like get_preferences — same protocol family, different primary value.

Why is Team a flat price instead of per seat?

Many coding assistants bill per user per month. Our Studio tier is priced as a single workspace for small teams (up to 15 users in product copy, £39/mo here). Compare total cost accordingly.
